Research and Design Project

  • Short term parking garage-scale occupancy study | Capstone Project
    • This project aimed to track the number of vehicles that entering and leaving the garage, find the available spots, and report the data to users.
    • The group placed RFID tags underneath the speed bumps and read the data from the RFID reader. By calculating the time stamp when the reader last sees the RFID tags, the program, which wrote in Python, knew if a car entered or left the garage.
    • The team used a WISP Camera, which is a battery free chip with micro-controller and a camera on it, to take pictures of the garage. I used decision tree based learning algorithm to find the available spots.
    • The team created a website running on Amazon Web Service using HTML, CSS, JavaScript , and adopted SQLite to manage the data report from the program on a local computer.
  • Robomasters Competition
    • The Robomaster competition required teams to build manually controlled and autonomous robots for shooting plastic bullet towards robots from other teams on a designated competition field.
    • The University of Washington team designed four manually controlled robots and one autonomous robot. I participated in designing the control system of the chassis and the shooting system for all robots on STM32 development boards.
    • The team won the third prize in the final competition.
  • Real-time Game Between Two FPGA Boards
    • The team designed and implemented a real-time UART communication protocol by Verilog on DE1-SoC boards and designed the graphic user interface(GUI) by Verilog and output by the on-board VGA module.
  • Social Network Visualization
    • The algorithm, which was inspired by K-means algorithm and force-directed graph drawing algorithm, can assign each vertex in a social network to a geographic coordinate in a given coordinate plane by iteration from a edge list file. Fianlly, the vertices which have edges will be geographically close to each other. I am currently writing the paper of this project.
